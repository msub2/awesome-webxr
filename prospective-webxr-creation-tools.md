# Prospective WebXR Creation Tools

This list includes any tool where there has been some discussion/planning to incorporate WebXR support.

|Name|Type|Issue?|PR?|Notes|
|--|--|--|--|--|
|[Armory](https://github.com/armory3d/armory)|Engine|[1587](https://github.com/armory3d/armory/issues/1587)|No|Prior versions had WebVR support, but even that no longer works due to changes in related things used by the engine|
|[Bevy](https://bevyengine.org/)|Engine|[115](https://github.com/bevyengine/bevy/issues/115)/[23](https://github.com/awtterpip/bevy_oxr/issues/23)|[2319](https://github.com/bevyengine/bevy/pull/2319)|Work stalled in main some years ago, but OpenXR support is now possible through a plugin, which has an open issue on its repo for WebXR support.|
|[StereoKit](https://stereokit.net/)|Library|[345](https://github.com/StereoKit/StereoKit/issues/345)|No|Not likely anytime soon, current rendering is built for Vulkan.|
